# POST /v1/memory/replay/run/end

Use this endpoint to close an explicitly instrumented replay run.

## What it is for

1. ending replay capture cleanly
2. finalizing run-level replay state
3. completing low-level replay instrumentation

## Minimal request

```bash
curl -sS "$BASE_URL/v1/memory/replay/run/end" \
  -H 'content-type: application/json' \
  -d '{
    "tenant_id":"default",
    "scope":"docs_replay",
    "run_id":"replay_run_001",
    "status":"completed"
  }' | jq
```

## Example response

```json
{
  "ok": true,
  "run_id": "replay_run_001",
  "status": "completed"
}
```

## Choose this route when

Use `run/end` to close a step-instrumented replay run. It is the low-level closeout path, not the normal place to start replay.
