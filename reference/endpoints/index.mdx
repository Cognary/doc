# Endpoint Reference

This is the endpoint-level reference for the highest-value Aionis routes.

It starts with the routes most likely to matter during evaluation and first integration.

## Fast route-family map

```text
memory/write -> recall_text -> context/assemble -> rules/evaluate -> tools/select
handoff/store -> handoff/recover
replay/runs/get -> playbooks/compile_from_run -> playbooks/candidate -> playbooks/run
```

Read the endpoint pages in that order if you want the shortest path from first proof to deeper governed execution.
